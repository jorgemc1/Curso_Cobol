# Curso de GNUCobol
Projeto pessoal de um curso de GNUCobol

## 🚀 Como Rodar

# Compilando em linha de comando:
- cobc -x programa.cob -o programa

# Rodando o programa no Windows
prog1

# Rodando o programa no Linux
$ ./prog1

# Compilação com padrão IBM COBOL
cobc -x -Wall -std=ibm programa.cob -o programa

## 📌 Resumo Comparativo de compilação

| Opção | SEM -std | -std=ibm |
|---|---|---|
| Flexibilidade | Alta | Menor |
| Rigor do compilador | Médio | Alto |
| Aproximação com mainframe | Baixa | Alta |
